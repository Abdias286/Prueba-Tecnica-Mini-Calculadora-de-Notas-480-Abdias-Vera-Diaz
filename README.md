--Hola, Buenos dias, soy Abdias y este es mi codigo para la prueba tecnica donde realizo una calculadora de notas usando Java en Netbeans--

El objetivo de este ejercicio es demostrar mi logica de programacion, validaciones y una aproximacion a la Programacion Orientada a Objetos.

--- Â¿Que hace este programa?
1) Ingresar un numero de asignaturas de un alumno del 1 al 10.
2) Introducir la nota de cada asignatura y validar que este entre 0 y 10.
3) Calcular la media de todas las notas.
4) Contar cuantas asignaturas estan suspensas y mostrar la cantidad.
5) Mostrar todas las notas introducidas.
6) Indicar el resultado final entre APROBADO O SUSPENDIDO.

----- Lo que me parecio facil ---
La parte mas facil fue el uso de los tipo de datos, bucles, validaciones y condicionales porque me gusta practicarlo mucho por mi cuenta y ademas lo solemos reforzarlo mucho en clase
por lo cual se me hizo fluido el saber como usarlos de la manera correcta.


----- Lo que me costo mas ----
Lo que me costo mas fue el uso de Programacion orientada a objetos, especialmente el concepto del "encapsulamiento" ya que aun no lo vemos en clase
pero me encanta el aprendizaje autodidacta y ya habia revisado por cuenta propia. Aun asi
mi conocimiento es basico y este fue un gran ejercicio para reforzarlo y aprender mas. Mi intencion al usarlo era hacer
que el codigo sea mas limpio, profesional, modular y facil de ampliar para simular una situacion de lo mas real de como se trabaja un proyecto real.

Espero que haya sido de su agrado el codigo, muchas gracias por tomarse el tiempo de revisarlo y leerlo, fue un ejercicio muy util y disfrute mucho hacerlo.
